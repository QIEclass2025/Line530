포켓몬 지뢰찾기 게임

포켓몬 세계관을 활용한 지뢰찾기 업그레이드 게임입니다.
PokeAPI에서 불러온 포켓몬 스프라이트와 한글 이름을 사용해, 지뢰 대신 포켓몬을 피해 나가는 방식으로 플레이합니다.

게임 특징

포켓몬이 지뢰 역할을 하는 지뢰찾기 게임

PokeAPI 연동으로 1세대 포켓몬 이미지 & 한글 이름 사용

보드 크기, 지뢰 수, 타일 크기, 힌트 개수 직접 조절 가능

힌트 기능으로 안전한 칸 자동 오픈

타이머 모드: 시간 제한을 초 단위로 설정 가능

승리/패배 시 모든 포켓몬(지뢰) 자동 표시

커스텀 게임오버 팝업: 포켓몬 이미지 + 안내 메시지 + 다시 시작 버튼 제공

요구사항

Python 3.8 이상

필요한 패키지:

tkinter (표준 GUI 라이브러리)

requests (API 통신)

Pillow (이미지 처리)

설치 방법

이 저장소를 클론하거나 ZIP 파일로 다운로드합니다.

git clone <이_프로젝트_저장소_URL>
cd <프로젝트_폴더>


필요한 패키지를 설치합니다.

pip install requests pillow


(선택) uv 패키지 매니저를 사용할 경우:

uv run "포켓몬 지뢰찾기.py"

게임 실행 방법

포켓몬 지뢰찾기.py 파일을 실행합니다.

python "포켓몬 지뢰찾기.py"


상단의 설정 버튼을 눌러 보드 크기, 지뢰 수, 타일 크기, 힌트 개수를 원하는 대로 조정할 수 있습니다.

(선택) 타이머 설정 버튼을 눌러 제한 시간을 초 단위로 입력합니다.

0 또는 취소 → 타이머 끔

양의 정수 입력 → 그 시간으로 카운트다운 시작(첫 클릭 시점부터)

보드에서 칸을 클릭하며 게임을 진행합니다.

좌클릭: 칸 열기

우클릭: 깃발 표시/해제

모든 지뢰가 아닌 칸을 열면 승리, 포켓몬(지뢰) 칸을 열거나 시간이 0이 되면 패배입니다.

추가 기능
힌트 시스템

상단의 힌트 사용 버튼으로 사용 가능

아직 열리지 않은 칸 중에서 지뢰가 아닌 안전한 칸 하나를 자동으로 열어줍니다.

사용 가능 횟수는 설정창에서 조정할 수 있습니다.

힌트가 0이 되면 버튼이 비활성화됩니다.

타이머 모드

상단에 타이머: 끔 또는 남은 시간: XX초 형식으로 표시됩니다.

타이머 설정 버튼을 눌러 제한 시간을 직접 입력할 수 있습니다.

예: 60 → 60초 제한

타이머는 첫 번째 칸을 클릭한 순간부터 카운트다운을 시작합니다.

시간이 0초가 되면:

모든 포켓몬(지뢰)이 분홍색 배경으로 표시되고

“시간 초과” 패배 팝업이 나타납니다.

게임 설정

상단의 설정 버튼으로 별도의 설정창을 열 수 있습니다.

다음 항목을 조절할 수 있습니다.

항목	설명	범위 (기본값)
보드 크기 (NxN)	한 변의 길이	5 ~ 20 (기본 10)
포켓몬(지뢰) 수	지뢰 개수	보드 크기에 따라 자동 제한
타일 크기 (픽셀)	버튼 한 칸의 픽셀 크기	18 ~ 48 (기본 24)
힌트 개수	게임당 사용 가능한 힌트 횟수	0 ~ 10 (기본 3)

보드 크기와 첫 클릭 보호(3×3 영역 무지뢰)를 고려해 배치 가능한 지뢰 최대 개수를 자동 계산하며,
그보다 많이 입력하면 경고 메시지와 함께 최대치로 조정됩니다.

승리 / 패배 연출

승리 시

지뢰를 모두 깃발로 표시하지 않아도, 지뢰가 아닌 칸을 모두 열면 승리 처리

모든 포켓몬(지뢰)을 연한 노란색 배경으로 표시

패배 시 (지뢰 클릭)

클릭한 지뢰 칸은 붉은색 배경으로 강조

다른 지뢰들은 분홍색 배경으로 표시

큰 포켓몬 이미지와 함께 게임오버 팝업 표시

시간 초과 시

모든 지뢰를 분홍색 배경으로 표시

“시간 초과” 전용 메시지가 별도로 출력
